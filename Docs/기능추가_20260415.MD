[UI 기능 추가 또는 변경]
1. 데스크탑 응용프로그램 타이틀바 가려지게 하는 것을 기본값으로 설정. 안보이게함.
2. 단축키로 응용프로그램 타이틀바 보이게 가려지게 함. 
   cmd + 1 누르면 타이틀바 보이기
   cmd + 2 누르면 타이틀바 가려기
3. 화면 크기를 단축키로 조절하기 
   cmd + 5 누르면 20% 화면 크기 커짐
   cmd + 4 누르면 20% 화면 크기 작아짐
4. 응용프로그램의 타이틀바 DotCamera 표시를 AtomOppa 로 수정 

5. 추가 수정
최대 최소 닫기 버튼은 메뉴에 숨겨주고, 마우스로 클릭해서 이동되게 만들어어. 마우스 왼쪽버튼으로  창을 클릭 선택하면 위치 이동 가능하게 한다. Devtool 창이 기본적으로 보이게되어 있는데 이것도 가린다. 구현되면  테스트 가능하게 바로 실행.

6. 기능 다시 수정
-최대/최소/닫기 버튼을 숨기기
-마우스 오른쪽 클릭하면 팝업으로 창 최대/최소/닫기 메뉴 기능 제공
