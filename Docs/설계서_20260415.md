## 3. 모듈 설계

### 3.1 electron_main.js (Main Process)

| 함수 | 책임 |
|------|------|
| `checkAndRequestCameraPermission()` | 카메라 권한 요청 |
| `allowMediaDevices()` | 세션 권한 설정 |
| `createWindow()` | BrowserWindow 생성 |

### 3.2 dotcamera.html (Renderer Process)

| 요소 | 책임 |
|------|------|
| `<video id="video">` | 웹캠 캡처 |
| `<canvas id="c">` | SwissGL 렌더링 |
| `DotCamera 클래스` | 필터 알고리즘 |

---

## 4. 데이터 흐름

```
[웹캠] → [Video Element] → [SwissGL Texture] → [Dot Filter]
                                              │
                                              ▼
                                      [Screen Display]
```

### 4.1 필터 알고리즘

1. **luminosity**: 명도 계산
2. **pyramid**: 다중 스케일 분석
3. **gradient**: 기울기 계산
4. **particles**: 입자 이동 시뮬레이션
