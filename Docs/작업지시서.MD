## [2] 코딩 규칙

- **작성자**: `@>-- microdit` 로 기록

- **주석**: 수정하는 내용 최대한 주석으로 상세하게 설명

## [5] 빌드 및发行

### 버전 빌드

```bash
./build.sh
```

**작업:**
1. package.json 버전 patch 번호 1 증가
2. `npm run build` 실행
3. DMG 파일 생성 (DotCamera-x.x.x.dmg)

### 빌드 결과물

| 파일 | 용도 |
|------|------|
| `dist/DotCamera-x.x.x.dmg` | macOS 설치 파일 |
| `dist/DotCamera-x.x.x.dmg.blockmap` | 업데이트용 |

### 자주 사용하는 명령어

| 명령어 | 용도 |
|--------|------|
| `./build.sh` | 버전 증가 + 빌드 |
| `npm start` | 개발 모드 실행 |
| `npm run build` | 빌드만 수행 |
