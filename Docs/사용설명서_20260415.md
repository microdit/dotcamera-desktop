## 1. 프로젝트 개요

**AtomOppa-DotCamera**는 웹캠 영상을 실시간으로 도트 패턴 필터로 변환하여 보여주는 macOS 데스크톱 애플리케이션입니다.

---

## 2. 기능

- 웹캠 실시간 영상 캡처
- 도트 패턴 필터 적용 (SwissGL/WebGL2)
- 실시간 렌더링 (60fps 목표)
- Electron 기반

---

## 3. 사용 방법

### 3.1 실행

```bash
npm start
```

### 3.2 빌드

```bash
./build.sh
```

### 3.3 설치

1. `dist/DotCamera-x.x.x.dmg` 더블 클릭
2. `Applications` 폴더로 이동

---

## 4. 프로젝트 구조

```
AtomOppa-DotCamera/
├── electron_main.js
├── dotcamera.html
├── swissgl.js
├── package.json
├── build.sh
├── build/entitlements.mac.plist
├── dist/DotCamera-x.x.x.dmg
└── Docs/
```

---

## 5. 자주 사용하는 명령어

| 명령어 | 용도 |
|--------|------|
| `./build.sh` | 버전 증가 + 빌드 |
| `npm start` | 개발 모드 실행 |
| `npm run build` | 빌드만 수행 |

---

## 6. 문제 해결

### 화면이 흰색/검은색만 표시

- 카메라 권한 확인 (시스템 설정 → 카메라)
- 앱 재시작

### 빌드 오류

```bash
rm -rf node_modules dist
npm install
./build.sh
```
